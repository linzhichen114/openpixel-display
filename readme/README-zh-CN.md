# OpenPixel - The Display Protocol


## OpenPixel是什么？
OpenPixel是一个给CoolPotOS用的显示协议。\
它易于使用且具备高度可移植性。\
OpenPixel提供 C 语言基础库和 Lua 扩展库支持。\
查阅《[OpenPixel 库函数API表](https://linzhichen114.github.io/~openpixel/docs/libraries/table.html)》开始使用。
